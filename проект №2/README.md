# 🤖 SaveBot — бот для сохранения удалённых сообщений

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Открой `config.py`
2. Вставь токен от @BotFather:
   ```python
   BOT_TOKEN = "твой_токен_здесь"
   ```
3. Вставь свой Telegram ID (узнать у @userinfobot):
   ```python
   ADMIN_ID = 123456789
   ```

## Запуск

```bash
python bot.py
```

## Подключение бота к Telegram Business

1. Убедись что у тебя есть **Telegram Business** подписка
2. Зайди в **Настройки → Telegram Business → Чат-боты**
3. Найди своего бота и подключи его
4. Готово! Бот начнёт перехватывать удалённые сообщения

## Функции

| Функция | Бесплатно | Premium |
|---|---|---|
| Текстовые сообщения | ✅ | ✅ |
| Фото и видео | ❌ | ✅ |
| Файлы и голосовые | ❌ | ✅ |
| История хранения | 3 дня | 30 дней |

## Выдача Premium пользователю (для админа)

```
/premium 123456789
```
где `123456789` — Telegram ID пользователя
